# AI_Investment_Advisor

# 🌍 The Global Contagion Effect: Executive Strategy Hub

An interactive, web-based financial dashboard designed to explain complex macroeconomic concepts—specifically how United States political drama and tariff threats crash the Spanish stock market—using plain, accessible, Gen-Z friendly language.

Unlike traditional, boring financial reports, this hub uses a dark-mode **Cyberpunk/Glassmorphism UI**, interactive data visualizations, and auto-playing story sliders to keep users engaged from start to finish.

## ✨ Key Features

*   📖 **Interactive Storytelling:** Auto-playing, sleek image carousels that break down the 2008 Financial Crisis and how economic contagion spreads across the Atlantic.
*   🌍 **Bilingual Support:** Instant 🇬🇧 English / 🇪🇸 Spanish toggle built seamlessly into the navigation bar (no page reloads required).
*   📊 **Live-Action Market Hubs:** Tracks key benchmarks and corporate valuations for both the US (S&P 500) and Spain (IBEX 35).
*   🚨 **Drama Analysis:** Event-driven statistical mapping that proves exactly how much the Spanish market drops (T+3) after aggressive US political rhetoric.
*   🔮 **Forecast Simulator:** An interactive widget allowing users to adjust "Global Health" and "Political Risk" to predict future structural risk boundaries.
*   🧠 **Cognitive Bias Stress Test:** Exposes corporate "blind spots" by forcing optimistic models to face severe historical shocks (like 2008) and theoretical Black Swans (like a Global Pandemic).
*   💡 **Actionable Recommendations:** A final sliding carousel delivering top-tier strategic advice ("Prepare, Don't Predict") in easy-to-understand language.

## 🛠️ Tech Stack

**Backend:**
*   **Python 3.x**
*   **Flask** (Web Server & API routing)
*   **Pandas / NumPy** (Data processing and mathematical modeling)

**Frontend:**
*   **HTML5 / CSS3** (Custom UI with Neon variables, Glassmorphism cards, and responsive CSS Grid/Flexbox)
*   **Vanilla JavaScript** (Carousel logic, Intersection Observers for scroll animations, API fetching, and Language switching)
*   **Plotly.js** (Interactive, responsive financial charting)
